El **modelo de Von Neumann** describe la arquitectura clásica de los ordenadores. Propone que datos e instrucciones se almacenan en la misma memoria. --- # docs/pagina2.md # Página 2 – Componentes del Modelo de Von Neumann Los componentes esenciales son: - CPU - Memoria - Entrada/Salida - Buses --- # docs/pagina3.md # Página 3 – Comparación con Harvard La arquitectura Harvard separa la memoria de datos y la de instrucciones, a diferencia del modelo de Von Neumann.
